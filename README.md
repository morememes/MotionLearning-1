# MotionLearning
GUI interface for processing video.


# Some Facts
- This program works correctly only with .mp4 files.
- In 'Settings' you can also add audio changing.


# Requirements
- Python 3.6
- PyQt5
- numpy
- opencv-python
- imutils
- librosa
- os, subprocess, shutil, time, sys (standart modules)


# Updates
